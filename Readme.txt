Calculates number of standard rectangle/square sheets required for only one piece of rectangle/square metal plate. Returns number of market standard sheets to use.

The code takes parameters like this: 
rectangle(area_dimension, standard_plate_dimension)
rectangle([23.0, 1.5], [1.2, 2.4])

The cone() function calculates number of standard sheets to develop a cone. It is unfinished but works for certain sizes of cones.
